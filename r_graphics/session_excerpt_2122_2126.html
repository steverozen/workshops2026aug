Here's R/ggplot2 code that reproduces the seaborn plot. Assuming you saved the data as CSV:

```python
fmri.to_csv("fmri.csv", index=False)
```

Then in R:

```r
library(tidyverse)

fmri <- read_csv("fmri.csv")

# seaborn lineplot default aggregates: mean + 95% bootstrap CI
ggplot(fmri, aes(x = timepoint, y = signal,
colour = region, linetype = event,
fill = region)) +
stat_summary(fun.data = mean_cl_boot, geom = "ribbon",
alpha = 0.2, colour = NA) +
stat_summary(fun = mean, geom = "line", linewidth = 0.8) +
scale_colour_manual(values = c(parietal = "#4C72B0", frontal = "#DD8452")) +
scale_fill_manual(values = c(parietal = "#4C72B0", frontal = "#DD8452")) +
scale_linetype_manual(values = c(stim = "solid", cue = "dashed")) +
theme_gray(base_size = 12) +
theme(
panel.background = element_rect(fill = "#EAEAF2", colour = NA),
panel.grid.major = element_line(colour = "white"),
panel.grid.minor = element_line(colour = "white", linewidth = 0.3),
legend.key = element_rect(fill = "#EAEAF2")
) +
labs(x = "timepoint", y = "signal", colour = "region",
linetype = "event", fill = "region")
```

Notes:
- `stat_summary(mean_cl_boot)` matches seaborn's default 95% bootstrap CI. For a faster approximation use `mean_cl_normal` (normal-theory CI) or `mean_se` (±1 SEM, narrower).
- The two manual scales for colour and fill use the seaborn "deep" palette hex codes so the ribbons match the lines.
- `theme_gray` plus the panel-background tweaks approximate seaborn's `darkgrid` style. If you have the `ggthemes` package, `ggthemes::theme_solarized()` or the `seaborn` R package's `theme_seaborn()` are alternatives, but the inline theme above avoids the extra dependency.
